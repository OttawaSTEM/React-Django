# Prettier format
Install "Prettier - Code formatter" extension in VSCode
```
    $ yarn format
    $ yarn format --fix
```

Then run following commands in frontend dir
```
    $ yarn lint
    $ yarn lint --fix

    $ yarn format
    $ yarn format --fix

    $ yarn install
    $ yarn start
```
React app available at `http://localhost:3000/`
